# evil-winrar

## 介绍

evil-winrar是winrar cve-2023-38831漏洞利用和社会工程学攻击框架,支持exp生成,邮件发送和下载链接生成

## 下载

```
git clone https://github.com/youmulijiang/evil-winrar.git
cd evil-winrar
python evil-winrar.py
```
